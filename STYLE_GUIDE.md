# rndvx Style Guide

Design system for building consistent scheduling experiences. Exported from Figma.

---

## Typography

### Fonts

| Font | Usage |
|------|-------|
| **Space Grotesk** | Headings and display text |
| **Raleway** | Body text, labels, and UI elements |

### Scale

| Token | Size | Weight |
|-------|------|--------|
| Heading 1 | 36px | Bold |
| Heading 2 | 30px | Bold |
| Heading 3 | 24px | Semibold |
| Heading 4 | 20px | Semibold |
| Large Text | 18px | Semibold |
| Body Medium | 16px | Medium |
| Body Regular | 16px | Regular |
| Small Text | 14px | Regular |
| Extra Small Text | 12px | Regular |

## Colors

### Primary Colors

White text overlay on these backgrounds.

| Name | Hex | RGB | Usage |
|------|-----|-----|-------|
| Lime Green | `#9BD770` | `rgb(155, 215, 112)` | Primary backgrounds, success states |
| Coral Pink | `#FF705D` | `rgb(255, 112, 93)` | Primary backgrounds, action states |

### Accent Colors

For highlights, badges, and secondary actions.

| Name | Hex | RGB | Usage |
|------|-----|-----|-------|
| Sky Blue | `#5EACFF` | `rgb(94, 172, 255)` | Highlights, interactive elements |
| Lavender | `#D4B8E8` | `rgb(212, 184, 232)` | Subtle highlights |
| Yellow | `#F5E211` | `rgb(245, 226, 17)` | Attention, energy |

### Neutral Colors

| Name | Hex | RGB | Usage |
|------|-----|-----|-------|
| Cream | `#F5F1E4` | `rgb(245, 241, 228)` | Primary background |
| Secondary BG | `#E0DBCF` | `rgb(224, 219, 207)` | Secondary background, muted elements |
| Charcoal | `#1A1A1A` | `rgb(26, 26, 26)` | Primary text |
| White | `#FFFFFF` | `rgb(255, 255, 255)` | Text on primary colors, cards, overlays |

## Cards & Containers

| Type | Description |
|------|-------------|
| Event Card | White background with border, on cream background |
| Secondary Card | Secondary background (`#E0DBCF`) for less prominent content |
| Accent Card | Sky Blue (`#5EACFF`) for highlighted/featured content |
